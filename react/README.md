# UFO 1999 (React SPA)

This project is a React single-page application for the UFO 1999 site. The navigation loads pages without a full refresh and Chapter 2 is split into sections with next/previous navigation.

## Requirements

- Node.js 18+.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

Open the URL printed in the terminal.
